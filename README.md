
# 오름(ORM)
**등산과 친해지고 싶은 사람들을 위한 통합 플랫폼**

자신만의 등산 일지를 기록하고 공유하며, 함께할 친구를 찾아보세요!

## Features

- ℹ️ **정보 제공 기능**
    - 산 정보 제공 (개요 및 등산로 정보)
    - 기본 등산로 네비게이션 역할

- 👨‍👩‍👦 **커뮤니티 기능**
    - 모임 기능
    - 채팅 기능
    - 게시글 기능

- 📂 **사용자 발자국 등록**
    - 개인 등산 기록 제공 및 분석
    - Instagram용 사진 추출

- 🤖 **AI**
    - Personalized Hiking Time Prediction System Using Machine Learning
## Tech Stack



**Client:** Android

**Server:** Spring Boot, MySQL
